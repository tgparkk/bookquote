### 계정 삭제 (출시 블로커)
- `lib/features/account/account_deletion.dart` — `runAccountDeletionFlow(context, ref)`: ① 영구 삭제 경고 + 내보내기 권유 다이얼로그([내보내고 탈퇴]/[계속]/[취소]) → ② "탈퇴합니다" 타이핑 확인 다이얼로그 → dim `_DeletingDialog`(`PopScope(canPop:false)`) → `supabase.functions.invoke('delete-account')` → `supabase.auth.signOut()`(라우터 가드가 `/auth/login`으로). 실패 시 세션 유지 + "탈퇴 처리에 실패했어요" 토스트(중간 상태 방지).
- `supabase/functions/delete-account/index.ts` — JWT(Authorization 헤더)로 호출자 `getUser()` 확인 → service_role 클라이언트로 `auth.admin.deleteUser(user.id)`. `auth.users` 삭제 시 `quotes`·`user_books`·`profiles`가 `on delete cascade`로 함께 삭제(`cards`는 Stage 3 마이그레이션에서 cascade 챙길 것). `_shared/cors.ts` 재사용. **배포는 미완** — `npx --yes supabase functions deploy delete-account` 필요(Edge Function엔 `SUPABASE_*` 자동 주입이라 시크릿 설정 불필요). 미배포 상태에선 회원 탈퇴 시 invoke 404 → "탈퇴 처리에 실패했어요" 토스트.
