## 가장 중요한 발견 (가상 인터뷰에서)

**가상 시뮬레이션이라는 한계는 있지만**, 다음은 실제 인터뷰에서도 검증해볼 만한 가설:

### 🟢 강한 가설 (실제 인터뷰에서 확인할 것)
1. **"사진은 찍는데 다시 안 봄"** 페인이 핵심 — 모든 핵심 페르소나에서 발현
2. **타겟은 "인스타 활성 + 책 좋아하는 20–30대 여성"** — 명확
3. **친구 timeline은 페르소나마다 가치 인식 갈림** — V1에 메인으로 두지 말고 보조로

### 🟡 검증 필요한 가설
1. **친구 단톡방 공유**가 친구 follow보다 더 강한 viral 메커닉일 가능성 — 실제 인터뷰에서 "최근 단톡방에 책 인용구 보낸 적?" 깊게 파보기
2. **Tezza가 직접 경쟁자** — 본인이 1주 직접 사용해서 확인

### 🔴 새로 발견된 V1 후보 기능
- **책 표지 컬러 자동 추출 → 카드 팔레트 생성** — 페르소나 5(디자이너)의 매우 구체적 요청. V1에 넣으면 디자인 까다로운 사용자 묶을 수 있음
