책 사진을 폰 기능으로 복사 → 책귀에서 "붙여넣기"

앱 밖(iOS Live Text / 구글렌즈) → /quote/new 진입 시 클립보드 감지 배너   방금 구현 (PR2)

민지는 읽던 책의 한 페이지를 폰 카메라로 비춰 iOS Live Text로 마음에 든 문장을 길게 눌러 복사한다(앱 내장 OCR을 우리가 다시 만들지 않는다 — OS가 이미 잘함). 책귀로 돌아오면 인용구 입력 화면 상단에 "📋 클립보드에 텍스트가 있어요" 배너 — 붙여넣기 한 번.

민지: 카메라 앱에서 문장 길게 눌러 복사 → 책귀 열기 → "붙여넣기" 탭. 본문에 문장이 채워진다.
  • 클립보드는 Clipboard.hasStrings()로 알림 없이 존재만 확인 → "붙여넣기" 누를 때만 실제 내용을 읽음(프라이버시)
  • 직접 타이핑도 물론 가능. 작성 중인 내용은 1초마다 로컬에 임시저장 — 앱이 죽어도 복구

책 연결 — 검색 시트 (Stage 1 그대로 재사용)

"책 연결" 탭 → showBookSearchSheet 모달 (알라딘 검색 + 캐시)   구현됨

"책 연결"을 누르면 책 검색 시트가 올라온다 — 서재에 책 추가할 때 쓰던 그 시트. 제목을 치면 캐시(이전 검색)와 알라딘 결과가 같이. 책을 고르면 시트가 닫히고 입력 화면에 책이 붙는다.

민지: "미드나잇 라이브러리" 검색 → 결과 탭. 인용구 입력 화면으로 돌아오니 책 카드가 채워져 있다. 페이지 "132" 입력, 무드 "위로" 칩 탭.
  • 책을 못 찾아도 인용구 저장은 됨(나중에 연결). 시트엔 ISBN 직접 등록 출구도 (PR4에서 보강)
  • 시트를 다녀와도 본문·페이지·무드는 그대로(모달이라 입력 화면 state 안 건드림)

카드 에디터 — 표지색 자동 팔레트가 이미 적용된 채로

/quote/:id/card   설계만 (Stage 3) — 지금은 인용구 저장은 되고 화면은 스텁

"카드 만들기 →"를 누르면 인용구가 저장되고 카드 에디터로. 들어가자마자 그럴듯한 카드가 그려져 있다 — 이 책 표지에서 뽑은 색이 자동 적용된 상태(차별화 ①). 5종 템플릿이 가로 한 줄로, 각 칸은 와이어프레임이 아니라 지금 그 인용구·표지로 실제 렌더된 미니 카드. "색 변경 UI"를 따로 안 만든다 — 표지 5스와치 줄이 곧 컬러 패널.

민지: 비율을 9:16(스토리)으로 두고, 마음에 드는 템플릿 1탭. "공유" 탭.
  • 언두 항상 있음(비율·템플릿 전환까지). 더블탭 숨김 같은 함정 없음 — 한 번 탭으로 다 됨
  • 흑백 표지 책이면 Ink-Paper-Copper 기본 톤으로 폴백. 표지 없는 책이면 "표지 발췌" 템플릿만 비활성

공유 — 카카오톡 단톡방으로 한 번에

"공유" → 모달 시트 (V1: share_plus OS 시트)   설계만 (Stage 3/4)

"공유"를 누르면 시트가 올라온다 — 1순위가 카카오톡 단톡방으로 보내기. 인스타 스토리(9:16 자동)·이미지 저장도. 사진 저장 권한을 거부해도 공유는 그대로 — 어디서도 막다른 골목 없음.

받는 쪽 — 단톡방에서 카드 탭 → "내 서재에 담기" 한 번

deep link io.github.tgparkk.bookquote://book/:id?from=share → /book/:id   설계만 (Stage 4 — 핸들러 일반화 필요)

단톡방의 친구 수연(23, 대학생)이 카드를 탭한다 → 책귀가 열리며 그 책 정보 + 민지가 보낸 인용구. "내 서재에 담기" 한 번이면 자기 서재에 들어간다. 책귀가 없으면 책 정보를 먼저 보여주고 → "담기" 누르면 로그인 → 돌아와서 담기. 이게 입소문의 마지막 1cm(K-factor).

수연: 카드 탭 → "📚 내 서재에 담기" → 담김. 며칠 뒤 이 책을 사서 자기 인용구를 만들어 또 단톡방에…
  • 북적북적이 "홈 화면을 우연히 캡처해 인스타에 올린다"로 1M 유저를 만든 그 등식을, 우리는 deep link 메커닉으로 의도 설계한다

다시 내 홈·서재 — 쌓이고, 무드로 다시 본다

/ 피드 · /library(책 ↔ 인용구 세그먼트)   설계만 (PR3·PR4)

며칠 뒤 민지가 홈을 열면 모은 인용구가 시간순으로 쌓여 있다. 각 항목에서 바로 [카드 만들기]. 서재 탭은 "책 ↔ 인용구" 세그먼트 — 책 보기, 또는 무드별로 인용구 다시 보기("위로 12 · 먹먹 8 · 새벽3시 5"). "사진은 찍는데 다시 안 봄" 페인의 답을 테마 단위 다시 보기로(차별화 ④).
